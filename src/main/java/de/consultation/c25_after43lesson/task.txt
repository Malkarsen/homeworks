Упражнение 1: «Универсальный приют для животных» (Animal Shelter)
Задание
	1.	Создайте абстрактный класс Animal, который содержит поля name
	(имя животного) и, например, age (возраст),
	а также абстрактный метод makeSound().
	2.	Создайте класс Dog, который наследуется от Animal.
	Пусть он определяет метод makeSound() так, чтобы выводить в консоль «Woof!».
	3.	Создайте класс Cat, который тоже наследуется от Animal.
	Метод makeSound() должен выводить «Meow!».
	4.	Создайте обобщённый класс AnimalShelter<T extends Animal>.
	В нём должен храниться список List<T> для приюта.
	Методы, которые нужно реализовать:
	•	admitAnimal(T animal): добавить животное в приют;
	•	releaseAnimal(String name): выпустить животное с данным именем из приюта
	 (или вернуть null, если такого нет);
	•	getAnimals(): вернуть список всех животных (только для чтения).
	5.	Проверьте работу приюта:
	•	Создайте AnimalShelter<Dog> и добавьте несколько собак;
	•	Создайте AnimalShelter<Cat> и добавьте несколько котов;
	•	Попробуйте «выпустить» какое-то животное по имени;
	•	Выведите всех оставшихся в приюте животных, вызовите для каждого
	makeSound().
Подсказки
	•	Обратите внимание на ограничение типа T extends Animal,
	чтобы приют мог хранить только наследников Animal.
	•	Используйте коллекцию, например, ArrayList<T>, для хранения
	животных внутри приюта.
	•	При поиске животного по имени переберите список и сравните поле name.
