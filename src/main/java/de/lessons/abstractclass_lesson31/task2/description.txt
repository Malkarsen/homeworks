### Упражнение 2: Спортивные соревнования
1. Создайте абстрактный класс `Competition` с полем:
   - `String competitionName` — название соревнования.
   Добавьте абстрактный метод `startCompetition()` и обычный метод `showCompetitionInfo()`, который выводит название соревнования.
2. Реализуйте два подкласса:
   - `Marathon` — соревнование по бегу. Метод `startCompetition()` должен выводить: "Марафон начался! Участники бегут 42 км."
   - `SwimmingChampionship` — соревнование по плаванию. Метод `startCompetition()` должен выводить: "Чемпионат по плаванию начался! Участники плывут 100 метров."
3. В методе `main()` создайте объекты соревнований, выведите их информацию и начните соревнования с помощью соответствующих методов.
Пример ожидаемого вывода:
```
Название соревнования: Марафон
Марафон начался! Участники бегут 42 км.
Название соревнования: Чемпионат по плаванию
Чемпионат по плаванию начался! Участники плывут 100 метров.
