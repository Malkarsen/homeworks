Задание 4: Выставка с ограничением по стилю
Создайте класс ArtStyle, который представляет стиль искусства
(например, "Impressionism", "Cubism").
Затем создайте класс Artwork, который содержит поля
title, artist и style(типа ArtStyle).
Реализуйте класс Exhibition, который использует Generics с ограничением,
чтобы хранить только произведения искусства определенного стиля.
Добавьте методы для добавления произведения и вывода всех произведений на экран.
