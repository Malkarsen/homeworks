Задание 3: Галерея с ограниченным типом
Создайте класс ArtPiece, который представляет произведение искусства
с полем year (год создания). Затем создайте класс ModernArtGallery,
который использует Generics с ограничением, чтобы хранить только
произведения искусства, созданные после 1900 года. Реализуйте методы
для добавления произведения и вывода всех произведений на экран.
