Упражнение 2: Работа с массивом номеров комнат и список гостей
Задача:

1) Предположим, у вас есть массив комнат int[] rooms = {101, 102, 103, 201, 202}
и массив для хранения имён гостей String[] guests = new String[5]. Индексы в этих массивах соответствуют друг другу.

2) Напишите метод addGuest(int roomIndex, String guestName), который:
Проверяет, что roomIndex находится в пределах массива rooms (иначе выбрасывает ArrayIndexOutOfBoundsException).
Проверяет, что guestName не пустое и не null (иначе выбрасывает IllegalArgumentException).
Если все проверки пройдены, записывает имя гостя в массив guests по индексу roomIndex.

3) Напишите метод getGuest(int roomIndex), который:
Проверяет, что roomIndex находится в пределах массива.
Если в ячейке guests[roomIndex] нет гостя (т.е. там null), выбрасывает собственное исключение NoGuestException
(сделайте класс, унаследованный от Exception), сообщая, что в данной комнате нет гостя.
Возвращает имя гостя, если он есть.

4) В методе main:
Вызовите addGuest с корректными и некорректными параметрами (например, слишком большой индекс, пустое имя).
Вызовите getGuest для разных индексов, проверяя различные сценарии: гость есть, гостя нет, индекс вне диапазона.
Перехватывайте все исключения и выводите понятные сообщения пользователю.

Подсказки:
При работе с массивами обязательно проверяйте граничные условия.
Для собственного исключения NoGuestException не забудьте вызвать super(message) в конструкторе.
