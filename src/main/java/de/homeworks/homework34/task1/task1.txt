Упражнение 1: Простая проверка корректности данных о бронировании
Задача:

1) Создайте класс SimpleBooking с методом bookRoom(int roomNumber, String guestName).
2) Пусть метод принимает номер комнаты и имя гостя.
3) Если номер комнаты меньше или равен 0, выбрасывайте IllegalArgumentException с сообщением о некорректном номере комнаты.
4) Если имя гостя пустое или равно null, выбрасывайте IllegalArgumentException с сообщением о некорректном имени гостя.
5) В методе main протестируйте работу метода bookRoom с несколькими вариантами входных данных.
6) Оберните вызовы в блок try-catch, корректно обработайте возможные исключения и выведите на экран сообщения об ошибке.

Подсказки:
Используйте try-catch в методе main.
При перехвате исключений выведите на экран понятное сообщение (например, «Некорректные данные бронирования: …»).
